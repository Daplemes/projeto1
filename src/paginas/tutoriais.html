Tutorial

Para entender como um site é construído, é necessário saber do que ele é feito.
Tres linguagens serão usadas pra definir cada parte do site, assim como seu comportamento e sua aparencia.
Usaremos HTML para criar a estrutura da pagina, definindo os espacos e o que será apresentado em cada um deles.
Em seguida, definiremos o comportamento dos links botões e paginas usando JavaScript.
Por ultimo, para então termos um site com um bom melhor visual, usaremos o CSS para mudar a aparencia dos espaços criados.

Eu usei VSCode como editor de texto, mas se quiser usar outros, não terá problema.
Também fiz uso de outras ferramentas como o GitHub para versionamento. Mas também não é necessário.
